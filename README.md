### 1. Install

1. Download the files in this repository into a local folder or clone the GitHub repository to your local machine
2. Open Chrome ‚Üí go to `chrome://extensions/` (for other Chromium browsers the address might be different, typically `{browsername}://extensions/`)
3. Turn on **"Developer mode"** (top right)
4. Click **"Load unpacked"** ‚Üí select the extension folder on your local machine
5. Done! Look for the RSpace icon in your browser toolbar

### 2. Connect 

1. Click the RSpace icon
2. Enter your RSpace server URL (e.g., `community.researchspace.com`)
3. Enter your API key (Get it from: My RSpace ‚Üí My Profile ‚Üí API Key ([Read more](https://documentation.researchspace.com/article/v0dxtfvj7u-rspace-api-introduction#obtaining_your_api_key))
4. Click **Connect**

### 3. Start Clipping

1. Go to any webpage
2. Click the RSpace icon
3. Choose what to save:
   - **Selection** - text you selected in your browser window
   - **Full Page** - entire page (representation in RSpace might deviate)
   - **URL Only** - just the link and the page tile
   - **PDF** - üÜï save entire page as PDF file attached to document
4. Select a document (or create a new one) ‚ö†Ô∏è *currently only works with simple documents and not yet with Forms*
5. (Optional) Add any additional comments
6. Click **Save to RSpace**

That's it! ‚ú®
